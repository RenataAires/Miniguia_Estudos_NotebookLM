# Caderno temático com NotebookLM
## Engenharia de Prompts e IA aplicada ao desenvolvimento de software.
---
## Contexto
Este projeto foi desenvolvido como desafio prático da DIO com o intuito de utlizar o NotebookLM como ferramenta de aprendizagem ativa.
O tema em questão foi escolhido devido o crescimento no uso de modelos de linguagem no contexto da programação.

---

- Compreender os fundamentos da Engenharia de prompt;
- Explorar estratégias de prompting para extraçã ode conhecimento;
- Entender limitações e vieses de modelos de IA;
- Aprender técnicas de organização de estudos com IA;
- Criar um material reutilizável para futuras revisões.

### Prompt 3
"O que define a autoria humana em códigos gerados por IA?"

Resultado:

Os principais critérios que definem a autoria humana nesse contexto são:
1. Contribuição Criativa Significativa
2. O Padrão de "Concepção Humana" (Patentes)
3. Distinção entre Criação Assistida e Autônoma
4. Necessidade de Documentação e Auditoria

"Em resumo, a autoria humana não é definida pelo ato de "apertar um botão", mas pela direção intelectual e pelo refinamento crítico que transformam uma sugestão bruta da IA em uma solução técnica funcional e original."

---

### Prompt 4
"Quais são as principais técnicas para otimizar respostas de LLMs?"

Resultado:

Técnicas de Prompting:

- Zero-shot e Few-shot Prompting
- Chain-of-Thought (CoT) Prompting
- Tree of Thoughts (ToT)
- Self-Consistency (Autoconsistência)
- Prompt Chaining
- Generated Knowledge Prompting

Estratégias Avançadas e Integração:

- Retrieval Augmented Generation (RAG)
- ReAct e Reflexion
- Automatic Prompt Engineer (APE)
- Uso de Ferramentas e Raciocínio Automático

Além dessas técnicas, as fontes destacam a importância da Engenharia de Contexto (Context Engineering) e do Fine-tuning (ajuste fino) como métodos para otimizar ainda mais a interação e os resultados obtidos.

# Miniguia de Estudos

## O que é Engenharia de Prompts?

É a prática de estruturar instruções para modelos de IA com o objetivo de obter respostas mais precisas, contextualizadas e úteis.

---

## Principais Técnicas

### Zero-Shot Prompting
O modelo responde sem exemplos prévios.

### Few-Shot Prompting
São fornecidos exemplos antes da solicitação.

### Chain-of-Thought
O modelo é incentivado a explicar o raciocínio passo a passo.

### Role Prompting
A IA assume um papel específico para melhorar contexto e comportamento.
